# Code of Conduct

We are committed to providing a friendly, safe, and welcoming environment for everyone.

## Our standards

Examples of behavior that contributes to a positive experience:

- Be respectful and considerate.
- Listen and respond thoughtfully.
- Assume good intent and be clear about expectations.

Examples of unacceptable behavior:

- Harassment, discrimination, or personal attacks.
- Trolling, insults, or derailing conversations.
- Publishing private information without permission.

## Enforcement

Project maintainers may remove, edit, or reject contributions that violate this code of conduct.

## Reporting

If you experience or witness unacceptable behavior, please report it to the maintainers.
For sensitive reports, open a private GitHub Security Advisory in this repository.
